# Amazon-Clothing-Sales-Analytics-Excel-Project
 Data-Driven Insights from 25,000+ Amazon Clothing Transactions
This repository contains a complete Excel-based analytics project analyzing Amazon clothing sales data (2024–2025). The project demonstrates a full data analytics workflow—from cleaning messy transactional data to building interactive dashboards and delivering actionable business insights.

 What's Inside
Data Cleaning & Preparation: Missing value handling, duplicate removal, feature engineering, and pricing validation

Exploratory Analysis: Univariate, bivariate, and multivariate analysis of sales, returns, pricing, and customer behavior

Hypothesis Testing: 5+ business hypotheses tested (discount vs returns, delivery time vs ratings, premium brand performance, etc.)

Advanced Analytics: Cohort analysis, RFM customer segmentation, anomaly detection, delivery hotspot mapping

Interactive Dashboard: Excel dashboard with pivot charts, slicers, and key business metrics

Business Report: 5–8 actionable insights with strategic recommendations
